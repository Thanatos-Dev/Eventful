# Eventful v1.0.2

The Eventful mod adds more invasions \& random events to fill out certain points in progression and certain areas of the map! The goal is to provide vanilla feeling content to give more variation to Terraria!



This mod is not currently compatible with multiplayer. If you'd like to help with multiplayer programming, join the discord \& let me know!



[Discord](https://discord.gg/9SdjGzGxcW)

[YouTube](youtube.com/@EventfulMod)

[Wiki](https://terrariamods.wiki.gg/wiki/Eventful)



# Content

* 1 invasion
* 1 event
* 37 total items
* 8 weapons
* 2 accessories
* 8 enemies
* 1 NPC
* 1 armor set
* 1 vanity item
* 1 food item
* 11 tiles
* 2 music tracks
* Bestiary entries \& filters
